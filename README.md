Repository for OOI Related AGU Presentation by Landung Setiawan

## Local Development

1. Setup environment

    ```bash
    mamba env create -f .binder/environment.yaml
    ```

2. Activate environment

    ```bash
    conda activate ooi-agu
    ```

3. Run Jupyter Notebook

    ```bash
    jupyter notebook
    ```

4. Go to http://localhost:8888/
